# kotlin-cli-bazel-erath-sieve

## Description
Given a number N, print all prime numbers smaller than N.

## Tech stack
- kotlin
- bazel

## Docker stack
- l.gcr.io/google/bazel:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
